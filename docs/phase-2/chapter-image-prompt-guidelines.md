# Chapter Image Prompt Guidelines

## Purpose

Each chapter may contain up to 10 cinematic image prompts.

The prompts should preserve:

- story continuity
- character continuity
- emotional continuity
- location continuity
- wardrobe continuity

## Recommended Image Types

1. establishing shot
2. protagonist moment
3. supporting character moment
4. relationship beat
5. location image
6. object or clue image
7. conflict image
8. emotional shift image
9. climax image
10. closing cinematic image

## Prompt Structure

Each prompt should define:

- chapter number
- image number
- story moment
- involved characters
- location
- mood
- camera framing
- lighting
- cinematic tone
- continuity notes
- approval state

## Continuity Expectations

Prompts should preserve:

- character appearance
- emotional state
- timeline correctness
- approved wardrobe logic
- established visual tone

## Approval Guidance

Images generated from prompts should remain:

- draft
- candidate
- approved
- locked

until author approval is completed.
