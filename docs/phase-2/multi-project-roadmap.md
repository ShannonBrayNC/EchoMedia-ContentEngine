# Multi-Project Roadmap

## Critical Requirement

The platform must support multiple simultaneous projects and franchises.

The API, UI, Christina workflows, semantic memory, and releases must never assume a single active project.

## Required Capabilities

- project registry
- project selection
- persistent project metadata
- per-project releases
- per-project continuity memory
- per-project screenplay exports
- per-project semantic memory isolation

## Project Types

- novel
- sequel
- TV adaptation
- feature film
- commercial campaign
- franchise universe

## Immediate Priorities

1. Multi-project registry
2. UI project switching
3. Christina project-aware orchestration
4. end-to-end test harness
5. vector memory isolation

## Product Conversion Goals

### Manuscript to Screenplay

```text
manuscript -> screenplay -> Fountain export
```

### Manuscript to Movie

```text
manuscript -> screenplay -> runtime -> trailer suitability -> production package
```

### Manuscript to TV

```text
manuscript -> season map -> episodes -> release package
```

### Manuscript to Commercial

```text
story hook -> commercial script -> social clip package
```
