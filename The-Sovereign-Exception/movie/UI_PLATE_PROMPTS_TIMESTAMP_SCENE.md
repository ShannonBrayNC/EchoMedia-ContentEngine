# The Sovereign Exception - UI Plate Prompts for Timestamp Scene

## Purpose

This file converts the UI mockup spec into direct prompt language for creating clean static UI plates or motion-design starting points for the timestamp proof-of-concept scene.

## Global UI Rules

- official secure-government interface aesthetic
- black or near-black background
- white primary text
- pale green stabilization/action states
- amber pending/warning states
- red escalation/failure states
- restrained, realistic, legible
- no decorative sci-fi clutter
- no AI face, no hologram theatrics, no neon cyberpunk overload

---

## Plate 1 - Green Map Crisis State

**Purpose:** Pre-stabilization crisis image.

**Prompt:** Create a clean secure-government crisis interface showing a dark global map with black oceans, red maritime threat boundary, a white drone swarm path crossing contested water, amber diplomatic relay lines, red hospital routing failure nodes, a falling market contagion graph, and a red retaliatory posture indicator, all in a realistic official UI style.

- AEGIS GLOBAL STABILIZATION REPLAY

- RETALIATORY POSTURE: ELEVATING
- HOSPITAL ROUTING: DEGRADED

# Export Guidance

- preferred aspect ratio for plates: 16:9
- 4K preferred, 1080p acceptable
- all text must remain readable at presentation size
- prioritize clarity over visual complexity
- if animated later, keep motion soft and clinical
