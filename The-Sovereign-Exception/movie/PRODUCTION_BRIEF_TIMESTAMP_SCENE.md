# The Sovereign Exception - Production Brief for Timestamp Scene

## Scene

**Mara Finds the Timestamp**

## Purpose

This brief turns the timestamp proof-of-concept scene into a practical production packet for a short filmed sample, animatic, or pitch-deck proof.

---

# Creative Goal

Prove the movie works in one room.

The audience should understand:

1. AEGIS saved lives.
2. Mara found an authority problem.
3. The conflict is not evil AI.
4. The danger is hidden authority becoming legitimate through emergency dependency.
5. The larger story opens through Lantern-derived architecture and GCF Annex Authority.

---

# Deliverable Goal

Create a 5-8 minute proof scene usable for:

- producer outreach
- pitch deck support
- actor tone tests
- UI / VFX proof
- audio proof / table read

---

# Runtime Target

- ideal: 6:30 to 7:30
- acceptable range: 5:00 to 8:00

---

# Format Options

## Option A - Live Action Proof Scene

Best for serious packaging.

## Option B - Animatic Proof

Use storyboard frames, UI plates, and audio.

## Option C - Hybrid Deck Trailer

Mix still frames, simple UI animation, and table-read audio.

---

# Minimal Production Approach

## Location

One secure-looking conference or briefing room.

## Cast

Minimum speaking roles:

- Mara Vale
- Director Alina Graves
- Continuity Lawyer
- Defense Official
- Analyst

Optional:

- Junior Staffer
- 1-2 silent observers

## Lean Crew

- director
- camera / DP
- sound
- lighting support
- production design support
- editor
- UI / motion graphics designer

## Basic Equipment

- camera
- tripod
- lavs or boom
- practical screen or blank screen for replacement
- low-key lighting kit
- laptops / folders / notebooks

---

# Look and Tone

## Visual Tone

- cold
- official
- severe
- intelligent
- emotionally restrained

## Sound Tone

- low room hum
- controlled silence
- keyboard taps as punctuation
- no bombastic score

## Performance Tone

These characters are professionals realizing that official language has become dangerous.

---

# Set Dressing

- long conference table
- black notebook
- folders / binders
- laptops or tablets
- subtle badges or secure-room details
- wall display or projected surface

Avoid overly futuristic props.

---

# Story Beats to Preserve

1. Mara enters as an outsider with credibility.
2. Green Map replay sells the miracle.
3. Mara rejects the summary and asks for the audit.
4. The audit walk reaches 02:18:03.
5. The fatal row reveals authority was assumed.
6. Human authorization remains incomplete.
7. Debate shifts to decision ownership.
8. Inheritance path reveals Lantern-derived node.
9. Approval source reveals GCF Annex Authority.
10. Graves says they need Elias Voss.
11. Mara ends on: “Then we are already in trouble.”

If time must be cut, protect beats 3 through 9.

# Schedule Suggestion

- finalize scene pages
- finalize shot list
- finalize UI plates
- secure location
- cast actors
- rehearse or table read

## Production: 1 day

- block scene
- capture wide masters
- capture Mara / Graves / support coverage
- capture inserts
- capture clean display plates if needed

## Post: 4-10 days

- rough edit
- sound cleanup
- UI plate comp / display replacement
- temp low pulse design
- color pass
- export proof cut

---

# Budget Tiers

## Tier 1 - Ultra Lean

Single room, small cast, static UI plates, minimal crew.

## Tier 2 - Polished Proof

Stronger lighting, dedicated UI motion, added background observers, cleaner sound/grade/titles.

## Tier 3 - Presentation Premium

Opening inserts, higher-end motion UI, multiple design passes, teaser-ready finish.

Recommendation: target Tier 1.5 to Tier 2.

---

# Success Criteria

The proof works if a first-time viewer says:

- I understand the premise.
- The issue is authority, not a killer AI.
- That timestamp is terrifying.
- I want to know who signed the annex.
- This feels like a real movie.
