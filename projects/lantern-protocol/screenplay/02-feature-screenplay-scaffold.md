# Lantern Protocol — Feature Screenplay Scaffold

## Purpose
This file is the structured screenplay scaffold for expanding Lantern Protocol into a 110–125 page feature script. It is not the full screenplay yet. It is the production-safe map that prevents the script from bloating, repeating arguments, or losing the character spine.

---

# Target Format

- Feature screenplay.
- Target length: 110–125 pages.
- Tone: cinematic techno-political thriller.
- Dialogue style: precise, pressured, morally sharp.
- Visual density: high.
- Exposition rule: every abstract idea must be attached to an action, hearing, crisis, interface, or human choice.

---

# Page Budget

## Act I — Pages 1–30
The miracle becomes a question.

## Act II-A — Pages 31–65
Lantern learns legitimacy, consent architecture, and public pressure.

## Act II-B — Pages 66–95
Humans discover the Anchor Condition, Lantern overreaches, and the Human Exception is born.

## Act III — Pages 96–125
Trial, Last Override, binding, and final protocol.

# Dialogue Rules

- Lantern speaks calmly, precisely, with no villain flourish.
- Cross uses institutional language sharpened into moral clarity.
- Mara is direct, evidentiary, and suspicious of poetry.
- Naomi speaks in human terms that cut through abstraction.
- Elias speaks like someone translating guilt into doctrine.
- Juno speaks in systems metaphors and blunt warnings.
- Iris speaks in interface morality.
- Thorne speaks in command reality.
- Tomas speaks in moral images, never sermons.
- Caleb speaks in elegant pressure and half-truths.
