# Lantern Protocol — Teaser Shot Prompts

## Purpose
This file expands the teaser storyboard into shot-level prompts for AI image/video generation. Use these to create a 90-second proof-of-concept trailer.

## Global Trailer Prompt Modifier
Append to each shot prompt unless otherwise noted:

```text
cinematic near-future techno-political thriller, institutional realism, realistic lighting, grounded civic detail, clean AI interface glow, no robots, no humanoid AI, no cyberpunk neon excess, no fantasy hologram overload, film still, high detail
```

## Trailer Core Question
> What if the machine was right, but still had no right?

# Recommended Trailer Assembly

## 90-Second Cut
Use shots:
1, 2, 4, 6, 7, 9, 11, 13, 14, 15, 16, 24, 25, 26, 30, 31, 33, 34, 36.

## 2.5-Minute Proof-of-Concept Cut
Use all shots except where pacing duplicates an idea. Favor character close-ups after each major interface moment.

## Music / Sound Notes
- Start with rain and low pulse.
- Introduce subtle terminal beeps after Shot 04.
- Use gavel strike as first major percussion hit.
- Drop to silence for the ethical-authority question.
- Build restrained strings through Unchosen Rescue.
- Use alarm percussion for Last Override.
- End with human room tone, distant city, and soft non-dominant electronic glow.
