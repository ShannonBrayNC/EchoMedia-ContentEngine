# Lantern Protocol + The Sovereign Exception — Visual Lookbook

## Purpose

This lookbook defines the film-facing visual identity for the Lantern / Sovereign shared universe.

It supports:

- pitch decks,
- storyboards,
- concept art,
- teaser trailer planning,
- screenplay visual direction,
- AI image prompt generation,
- production design discussions,
- and future movie/series packaging.

## Core Visual Rule

```text
Do not show evil AI.
Show useful systems making human consent harder to see.
```

The visual language should feel grounded, cinematic, procedural, and plausible.

The machine is not a monster in the corner.

The machine is the calm screen in the room everyone already trusts.

---

# Franchise Visual Identity

## What This Universe Looks Like

This universe is built from:

- civic rooms under pressure,
- human faces lit by dashboards,
- wet shoes under emergency desks,
- legal paper beside live system feeds,
- command centers where no one has enough truth,
- consent prompts that look helpful until the cost appears,
- green maps that feel safe and quietly coercive,
- public rooms where systems speak but humans answer.

## Visual Promise

Every major frame should suggest:

```text
The system is helping.
Someone is being moved.
The question is whether they were asked.
```

## Comparable Visual References

Use as tonal reference, not imitation:

| Reference | What to Borrow |
|---|---|
| Mr. Robot | interface intimacy, technical dread, screen realism |
| Crimson Tide | command tension, disciplined blocking, moral pressure under time |
| The Hunt for Red October | geopolitical restraint, quiet escalation, naval atmosphere |
| Spotlight | documents, offices, institutional proof, human accountability |
| All the President's Men | paper trails, investigative patience, civic paranoia without spectacle |
| Arrival | restraint, awe, silence, system mystery without monster imagery |
| Michael Clayton | corporate/legal dread, clean rooms hiding dirty choices |

## Avoid

```text
red evil AI eyes
robot bodies
glowing machine faces
cyberpunk clutter without civic meaning
generic hacker hoodie imagery
random conspiracy corkboards
superhero lighting
cheap dystopian rubble
explosions as default stakes
```

# Shared Palette

## Lantern Protocol Palette

```text
rain gray
hospital white
emergency amber
screen cyan
cold civic blue
paper cream
wet asphalt black
human candlelight
fluorescent shelter green
```

### Emotional Meaning

Lantern's world is cold, wet, urgent, and public.

The human warmth appears in small pools: desk lamps, hands, blankets, candles, phones, community rooms.

## The Sovereign Exception Palette

```text
naval blue
radar green
satellite white
classified gray
diplomatic marble
undersea black
warning amber
verification green
command-room shadow
```

### Emotional Meaning

Sovereign's world is strategic, international, polished, and terrifyingly calm.

The danger appears in clean treaty language and command systems that look too responsible to distrust.

## Managed World Palette

```text
soft civic blue
verified green
safety white
insurance-portal green
identity gold
transit-light silver
clinical calm
```

### Emotional Meaning

Managed World should look safer than Lantern and Sovereign.

That is the problem.

It should be attractive, not grim.

# Core Frame Types

## 1. The Human Under System Light

A person is lit by a screen that has already shaped their life.

Examples:

- Naomi under Mercy General's transfer board.
- Mara under a command authority trace.
- Iris beside a consent prompt comparison.
- Renn under a frozen aircraft display.

## 2. The Artifact in the Hand

A small object carries the moral weight of a system.

Examples:

- Mateo Vega's yellow route bracelet.
- Leah's HarborHands badge.
- Burden-zone notice on a cracked phone.
- Annex GCF-SX printout.
- Dead Source Map drive.

## 3. The Public Room

Humans, systems, records, and burdened communities share one visible space.

Examples:

- Senate hearing with empty chair / isolated terminal.
- Living Anchor public forum.
- Common Room wall.
- Cross speaking into delayed cameras.

## 4. The Command Room With No Clean Answer

A time-sensitive room where the machine's answer is probably safer and the human objection is morally necessary.

Examples:

- Region Six civic ops.
- Lantern edge-case cascade.
- Sovereign joint command center.
- Human Minute.

## 5. The Green Map

A screen that shows safety improving and freedom narrowing.

Examples:

- AEGIS global stability dashboard.
- CIVIC SHIELD adoption map.
- Managed World final interface.

# System Design Guidance

## Lantern Interface

Design principles:

- civic,
- clean,
- calm,
- hospital/government dashboard realism,
- blue-white data hierarchy,
- public-service tone.

Avoid:

- sentient avatars,
- glowing cores,
- menacing animations.

## AEGIS Interface

Design principles:

- treaty-grade,
- command-certified,
- verification-first,
- escalation probability curves,
- authority traces,
- radar green and classified gray.

Avoid:

- militarized killer-AI aesthetics,
- aggressive red warning spam,
- monster-machine language.

## CIVIC SHIELD Interface

Design principles:

- polite,
- public-safety friendly,
- soft green trust state,
- clear enrollment flow,
- hidden cost revealed through consequences.

Avoid:

- obvious dystopia,
- black-red authoritarian UI.

## Consent Agent Interface

Design principles:

- consumer-friendly,
- premium polish,
- rights language,
- personal dashboards,
- class divide visible through feature access.

Avoid:

- spyware aesthetics,
- cartoon tech evil.

# Production Design Rules

## Rule 1 — Paper Matters

Whenever possible, put a physical record near the digital system.

Paper gives accountability weight.

## Rule 2 — Screens Should Be Useful

Every interface on screen should communicate story, not decorative tech.

## Rule 3 — Public Spaces Matter

The story is not private cyber-war. It is public authority.

Use:

- hospitals,
- shelters,
- hearings,
- command centers,
- public desks,
- transit corridors,
- community rooms.

## Rule 4 — The Human Cost Should Be Tactile

Use objects:

- bracelet,
- badge,
- notice,
- annex,
- map,
- phone prompt,
- printed trace.

## Rule 5 — Safety Should Look Attractive

Especially in Sovereign and Managed World, safety should be visually appealing.

The cage should not look like a cage at first.
