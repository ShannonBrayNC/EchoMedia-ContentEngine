# Lantern Protocol + The Sovereign Exception — Teaser Trailer Storyboard

## Purpose

This storyboard uses the 20 hero image prompts as the visual frame library for a teaser trailer package.

It supports:

- pitch deck development,
- concept art planning,
- teaser trailer scripting,
- proof-of-concept video generation,
- AI video prompt generation,
- and producer-facing visual presentation.

## Trailer Strategy

This file includes three trailer options:

1. **Shared-Universe Teaser** — sells the entire Lantern / Sovereign franchise.
2. **Lantern Protocol Teaser** — sells the civic-AI movie hook.
3. **The Sovereign Exception Teaser** — sells the geopolitical AEGIS movie hook.

## Core Trailer Rule

```text
The audience should want the system before they fear it.
```

Do not make the systems visually evil.

Show them working.

Then show what working costs.

# Shared-Universe Teaser

## Target Length

```text
90 seconds
```

## Trailer Thesis

```text
The future will not ask humanity to surrender freedom all at once.
It will offer relief.
```

## Tone

Quiet dread. Escalating pulse. No bombast until the final third.

The trailer should feel like a civic miracle slowly turning into a constitutional crisis.

## Music Direction

- Open with rain, hospital PA, distant thunder, soft civic alert tones.
- Add low pulse under the eight-second anomaly.
- Introduce muted command percussion during Sovereign crisis.
- Final section builds with strings and restrained electronic texture.
- End with sudden quiet under the Green Map.

## Voiceover Strategy

Use sparse voiceover from characters, not an external trailer narrator.

Primary voices:

- Naomi for human stakes.
- Mara for evidence.
- Thorne for command pressure.
- Iris for doctrine rupture.
- System text for calm menace.

# Trailer Production Notes

## What To Avoid

```text
No evil AI voice.
No robot imagery.
No explosions as primary selling point.
No generic hacker montage.
No overexplaining doctrine.
No trailer narrator saying the whole premise.
```

## What To Emphasize

```text
Useful systems.
Human artifacts.
Calm screens.
Impossible timestamps.
Legal phrases.
Command rooms.
Public rooms.
Green maps.
Refusal cost.
```

## Sound Design Motifs

| Motif | Use |
|---|---|
| Rain | Lantern civic crisis. |
| Hospital PA | Human pressure and public systems. |
| Board chime | System usefulness. |
| Low pulse | Authority anomaly. |
| Radar tone | Sovereign crisis. |
| Phone buzz | Annex clue. |
| Gate delay tone | CIVIC SHIELD cost. |
| System chime | Final unsettling calm. |

## Final Trailer Rule

```text
The teaser should not ask whether the system is evil.
It should ask whether being saved made permission optional.
```
