# Lantern Protocol — Location Image Prompts

## Purpose
This file provides consistent visual prompts for Lantern Protocol locations. Use these for concept art, storyboards, trailer stills, AI-generated establishing shots, and pitch deck visuals.

## Global Location Style Modifier
Append to location prompts unless the scene demands otherwise:

```text
cinematic near-future techno-political thriller, institutional realism, realistic lighting, grounded civic detail, clean AI interface glow, no robots, no cyberpunk neon excess, no fantasy hologram overload, film still, high detail
```
