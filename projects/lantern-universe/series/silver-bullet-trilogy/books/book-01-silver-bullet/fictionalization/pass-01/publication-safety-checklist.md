# Publication Safety Checklist

**Sprint:** SB-010  
**Book:** Book 01 — Silver Bullet  
**Status:** Active fictionalization controls

## Purpose
Create a publication-safety checklist for Book 1 before any excerpting, marketing copy, submission, or public release.

## Current Status
Book 1 is **not publication-facing**.

SB-010 provides controls for internal expansion. A later publication-safety review must confirm these controls have been applied to prose.

## Required Checks Before Public-Facing Use

## Character Checks

- [ ] No character maps directly to a real person.
- [ ] Major characters have invented biographies.
- [ ] Major characters have invented relationship histories.
- [ ] Dialogue does not reuse real correspondence.
- [ ] Physical descriptions do not identify real people.
- [ ] Professional roles are fictionalized.

## Setting Checks

- [ ] No exact real addresses.
- [ ] No exact real neighborhoods if identifiable.
- [ ] No exact real workplaces.
- [ ] No exact real court/government locations.
- [ ] Fictional geography is consistently used.
- [ ] Place names are invented or composite.

## Institution Checks

- [ ] No real customer names.
- [ ] No real employer names.
- [ ] No real agency names.
- [ ] No real case numbers.
- [ ] No exact real process labels.
- [ ] Institutional documents are invented.

## Chronology Checks

- [ ] No exact real event sequence.
- [ ] No exact dates tied to real-world personal events.
- [ ] Composite chronology is used.
- [ ] Timing serves fiction rather than recollection.

## Artifact Checks

- [ ] Messages are original fiction.
- [ ] Notices are original fiction.
- [ ] Meeting summaries are original fiction.
- [ ] Logs and screenshots are fictionalized.
- [ ] Metadata does not point to real systems.

## Trauma / Dignity Checks

- [ ] Jack remains agentic.
- [ ] Trauma is plot-active but not spectacle.
- [ ] Body limitation is dignified and not exploitative.
- [ ] Dark scenes change plot, character, or continuity.
- [ ] No scene uses suffering only for shock.

## Canon Checks

- [ ] Living Anchor remains human.
- [ ] Technology does not save truth by itself.
- [ ] Lantern mythology is not over-explained in Book 1.
- [ ] Book 1 does not fully resolve trilogy mystery.
- [ ] Book 2 handoff remains open.

## Legal / Reputation Checks

- [ ] No public-facing claim appears to accuse an identifiable real person.
- [ ] No institution can be reasonably identified as a real entity.
- [ ] No private record is reproduced.
- [ ] No real correspondence is paraphrased too closely.
- [ ] No marketing copy suggests true-crime or memoir positioning.

## Publishing Gate
The manuscript may only move to public-facing material after all checklist items are complete and a Producer Review refresh approves release.

## Producer Decision
Publication-safety checklist: **approved as required future gate**.
