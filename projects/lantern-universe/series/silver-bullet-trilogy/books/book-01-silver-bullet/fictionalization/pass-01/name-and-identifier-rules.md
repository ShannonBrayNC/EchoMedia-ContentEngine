# Name and Identifier Rules

**Sprint:** SB-010  
**Book:** Book 01 — Silver Bullet  
**Status:** Active fictionalization controls

## Purpose
Define naming, identifier, and metadata rules for Pass 2 expansion.

## Core Rule
No publication-facing manuscript material should include real-world identifiers, real case references, real customer names, real institutional labels, real addresses, or exact real correspondence markers.

## Character Naming Rules

1. Use fully fictional names.
2. Avoid names that closely resemble real people connected to the author, public records, customers, employers, legal matters, or family members.
3. Avoid initials that create obvious mapping.
4. Do not use exact professional titles that could identify a real person.
5. Give characters invented middle/context details only if useful.

## Chronology Rules

- Avoid exact real dates.
- Use relative time when possible.
- Use fictional timestamps only when needed for plot.
- Alter sequence enough to prevent one-to-one mapping.
- Combine multiple pressures into composite story events.

## Publication Safety Rule
Any identifier that can be searched, matched, or mapped to a real person or proceeding must be replaced before public-facing use.
