# Setting and Institution Map

**Sprint:** SB-010  
**Book:** Book 01 — Silver Bullet  
**Status:** Active fictionalization controls

## Purpose
Define fictional settings and institutions for Pass 2 expansion so Book 1 does not rely on identifiable real-world places, workplaces, courts, companies, agencies, or personal geographies.

## Core Rule
Use invented specificity. Replace real-world recognizability with fictional texture.

## Setting Replacement Rules

1. Do not use exact real addresses.
2. Do not use real court names, agency names, or case identifiers.
3. Do not use real workplace names or customer names.
4. Do not use exact real travel routes.
5. Use composite geography and invented local texture.
6. Preserve emotional truth without preserving identifying coordinates.

## Institutional Language Rules

Use generalized, fictionalized terms:

- review office
- account board
- workspace suspension
- access pause
- process summary
- context packet
- risk posture
- clarification request

Avoid direct real-world institutional jargon when it would create identifiable mapping.
