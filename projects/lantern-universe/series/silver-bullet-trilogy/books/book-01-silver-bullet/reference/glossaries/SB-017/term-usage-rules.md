# Book 1 Term Usage Rules

**Sprint:** SB-017  
**Book:** Book 01 — Silver Bullet  
**Status:** Canonical reference

## Purpose
Define how glossary terms should be used in future manuscript revision, scans, positioning, and release-gate work.

## General Rules

1. Use canonical fictional names consistently.
2. Do not substitute real names, real organizations, real agencies, real customers, real addresses, or real case labels.
3. Keep all artifact names fictional and internally consistent.
4. Use the glossary as the reference source for final safety scans.
5. Treat any non-glossary proper noun as requiring review.

## Character Rules

- Use `Jack Mercer` as the protagonist name.
- Use `Evelyn Mara Blackwood` for the full composite reference and `Evelyn Blackwood` in normal prose.
- Use full names at first formal reference, then natural shortened names where appropriate.
- Do not add biographical details that create real-person mapping risk.
- Keep supporting characters morally complex and fictional.

## Setting Rules

- Use `Briar County` as the primary fictional region.
- Use `Grayline Road` for Jack's neighborhood corridor.
- Use local settings only as fictional stagecraft.
- Do not add real address precision.

## Organization and Project Rules

- Use `Northstar Systems Group` for Jack's work environment.
- Use `Halden Ridge Migration Project` for the technical project.
- Use `CivicCloud Workspace` for workspace/access scenes.
- Use `Eastbridge Review Office` for process-pressure scenes.
- Do not use real employer, customer, agency, or system names.

## Artifact Rules

- Use `Sequence Record`, `Context Packet`, `Context Envelope`, `Echo Map`, and `Counter-Record` as core record terms.
- Use `Eastbridge Review Notice` and `Process Summary Memo` as fictional process artifacts.
- Use artifact names to preserve fiction distance.
- Do not make artifacts replace human witness.

## Scan Rule
Future scans should flag:

- any proper noun not in the glossary
- any organization label not in the glossary
- any document title not in the glossary
- any place name not in the glossary
- any artifact term that sounds documentary or real-world-specific

## Producer Decision
Term usage rules: **approved as canonical Book 1 reference**.
