# Book 1 Artifact Glossary

**Sprint:** SB-017  
**Book:** Book 01 — Silver Bullet  
**Status:** Canonical reference

## Purpose
Define the fictional document, record, message, and evidence-like artifacts used in Book 1.

## Core Artifacts

| Name | Type | Function | Usage Rule |
|---|---|---|---|
| Sequence Record | Fictional personal chronology | Jack's structured record of order, context, and contradiction | Use instead of documentary/legalistic terms where possible. |
| Context Packet | Fictional evidence/context bundle | Preserves artifacts with surrounding context | Use for Jack's assembled materials. |
| Context Envelope | Fictional interpretive wrapper | Defines source, limits, surrounding context, and possible misreadings | Use when a single artifact needs air. |
| Echo Map | Fictional pattern map | Tracks repeated phrases and social version drift | Use as Jack's internal story-movement map. |
| Counter-Record | Fictional witness package | Book 1 payoff artifact read by Mara Venn | Use carefully; do not make it a magical solution. |

## Formal / Process Artifacts

| Name | Type | Function | Usage Rule |
|---|---|---|---|
| Eastbridge Review Notice | Fictional notice | Formalizes work/process pressure | Keep invented and non-real-world. |
| Process Summary Memo | Fictional summary | Shows how partial records can change meaning | Keep original fictional prose. |
| Access Pause Notice | Fictional system/process notice | Shows CivicCloud access consequence | Use as invented workspace artifact. |
| Clarification Request | Fictional process message | Carries risk-language pressure | Keep generalized and fictional. |

## Social / Message Artifacts

| Name | Type | Function | Usage Rule |
|---|---|---|---|
| Evelyn Message | Fictional message artifact | Initial framing pressure | Must follow SB-013 voice rules. |
| Grayline Thread Capture | Fictional social-thread artifact | Shows earlier phrase timing and social echo | Use with fictional thread context. |
| Kendra Witness Note | Fictional witness fragment | Confirms timing issue with fear preserved | Keep partial and human. |
| Devin Version Account | Fictional verbal/social version artifact | Shows story compression | Use as witness-fragment, not proof shortcut. |
| Not-Sent Statement | Fictional temptation artifact | Captures false-mercy pressure | Keep as internal counterfactual. |

## Artifact Safety Rules

1. All artifacts must be original fiction.
2. Do not use real metadata, case labels, account IDs, or correspondence wording.
3. Do not make artifacts do the work of human witness alone.
4. Always preserve what an artifact proves and what it does not prove.
5. Keep artifact language internally consistent.
