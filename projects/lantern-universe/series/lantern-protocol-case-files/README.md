# Lantern Protocol Case Files

**Sprint:** PUB-003  
**Status:** Active series structure  
**Series Type:** Case-file anthology

## Purpose
Lantern Protocol Case Files is the shared-universe anthology layer for investigations, incidents, anomalies, witness records, and Living Anchor case material.

## Canon Role
This series connects individual case files to the broader Lantern Universe while preserving flexibility for standalone stories, novellas, short fiction, and experimental horror/mystery structures.

## Required Structure

```text
cases/
├── case-001-living-anchor/
└── case-###-<slug>/
```

## Required Governance
- Canon review
- Continuity review
- Case-file classification
- Producer Review
- Publishing package validation

## Scale Target
This structure should support dozens or hundreds of case files without disrupting trilogy-level canon.
