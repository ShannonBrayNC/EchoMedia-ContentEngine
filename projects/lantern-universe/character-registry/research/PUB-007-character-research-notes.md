## Registry Design Decisions

### Stable Character IDs
Each profile uses a stable `character_id` so future automation can link characters to timelines, books, relationships, visual bibles, and audiobook notes.

### Canon Status
Profiles use `canon-draft` or `concept-locked` so later Producer Review can promote them to `canon-locked` when fully reviewed.

### Fictionalization Controls
Sensitive profiles include explicit notes that they are fictionalized or composite. This helps protect the project from unsafe likeness or defamation drift.

### Trauma Continuity
Character profiles include trauma continuity so emotional and psychological consequences survive across drafts instead of disappearing between chapters.

### Collective Antagonist Support
The Widow Circle is treated as a collective character/organization bridge so SB-002 can expand it without losing registry discoverability.
