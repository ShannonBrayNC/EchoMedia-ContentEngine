# Lantern Universe Canon Bible

**Sprint:** PUB-006  
**Status:** Active canon system  
**Owner:** Lantern Publishing / EchoMedia Content Engine

## Purpose
The Canon Bible is the single source of truth for the Lantern Universe. It protects shared mythology, organizations, continuity, rules, terminology, and cross-series dependencies across Silver Bullet, The Voss Files, and Lantern Protocol Case Files.

## Canon Scope
This Canon Bible governs:

- Living Anchor mythology
- Narrative corruption rules
- Witness protocols
- Lantern organization rules
- Widow Circle canon
- Black Lantern mythology
- Cross-series continuity
- Glossary and naming standards

## Canon Rule
No manuscript, outline, case file, marketing package, audiobook package, or visual package may override this Canon Bible without a documented canon change and Producer Review.

## Locked Canon Anchors

1. **The Living Anchor is human.** Technology may preserve evidence, but only a human witness can serve as the true Living Anchor.
2. **The Voss Files is independent from Silver Bullet.** It is its own trilogy and reveals the deeper mythology behind Lantern.
3. **Lantern Protocol Case Files is the anthology layer.** It documents cases, anomalies, and Living Anchor incidents.
4. **Silver Bullet is grounded horror.** It connects to Lantern through narrative corruption, institutional failure, weaponized accusation, and witness preservation.
5. **Producer Review is mandatory for canon-changing work.**
