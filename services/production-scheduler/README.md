# Production Scheduler

## Purpose

Generate scheduling estimates for screenplay scenes and production sequences.

## Planned Inputs

- scene cards
- runtime reports
- trailer suitability reports
- production manifests

## Planned Features

- estimated shoot days
- scene batching
- location grouping
- runtime balancing
- production milestone planning

## Planned Outputs

```text
reports/production-schedule.json
```
