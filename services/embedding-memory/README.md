# Embedding Memory System

## Purpose

Provide persistent semantic memory across projects, characters, screenplay scenes, releases, and cinematic universes.

## Planned Memory Domains

- character memory
- relationship memory
- location memory
- screenplay memory
- visual continuity memory
- franchise continuity memory

## Planned Features

- vector indexing
- semantic recall
- narrative similarity search
- continuity drift detection
- franchise lineage analysis
- adaptation memory

## Suggested Technologies

### Vector Databases

- Azure AI Search
- PostgreSQL + pgvector
- Qdrant
- Pinecone

### Embedding Providers

- OpenAI
- Azure OpenAI
- local embedding models

## Christina Integration

Christina should use semantic memory retrieval to:

- summarize project history
- identify continuity risks
- recall prior screenplay decisions
- detect repeated story structures
- recommend trailer scenes

## Future Expansion

- cross-franchise memory
- cinematic motif memory
- adaptive production recommendations
- audience engagement memory
